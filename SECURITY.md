# Security Policy

## Reporting a vulnerability

This repo is a research umbrella with minimal runtime code. If you find a security concern in any code or example here, please report it privately rather than filing a public issue.

**Contact**: jeremy@intentsolutions.io

Acknowledgement target: 7 business days. Resolution timeline depends on severity and scope.

## Scope

- Vulnerabilities in code under `scripts/` (when present)
- Misleading methodology that could harm downstream users
- Credential leakage in committed files

## Out of scope

- Issues in constituent projects (`projects/` symlinks) — report those in their own repos
- Issues in upstream dependencies — report to those projects directly
